# 21天 Rust 实战教程

## 🚀 课程简介

欢迎来到"21天 Rust 实战教程"！本教程将带你从零开始学习 Rust 编程语言，通过每天一个主题，循序渐进地掌握 Rust 的核心概念和实践技能。

## 📚 课程特色

- **循序渐进**：从基础语法到高级特性，21天系统学习
- **实战导向**：每个主题都配有实际代码示例
- **中文讲解**：详细的中文注释和说明
- **项目驱动**：通过实际项目巩固所学知识

## 🎯 学习目标

完成本教程后，你将能够：
- 理解 Rust 的所有权系统和内存安全模型
- 掌握 Rust 的核心语法和特性
- 使用 Rust 开发实际应用程序
- 理解 Rust 的性能优势和适用场景

## 🚀 快速开始

1. **安装 Rust**
   ```bash
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```

2. **验证安装**
   ```bash
   rustc --version
   cargo --version
   ```

3. **开始学习**
   - 从 [第1天](./doc/day01.md) 开始
   - 每天完成一个主题
   - 动手实践代码示例

## 💡 学习建议

- **每天坚持**：保持学习的连续性
- **动手实践**：不要只看代码，要自己敲一遍
- **理解概念**：重点理解 Rust 的设计理念
- **项目练习**：通过实际项目巩固知识
